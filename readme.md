# 打字測驗程序

## 需求

1. 自定義題目
2. 倒計時測驗打字，測驗結束會有提醒跟鬧鈴
3. 點擊開始會先練習在進行測驗
4. 測驗結果會生成測驗報告，包含(字數統計、正確字數、錯誤字數、平均字數)

## 檔案結構

![](https://i.imgur.com/vE0ZcEH.jpg)

- data.pkl # 存放用戶自定義設定檔
